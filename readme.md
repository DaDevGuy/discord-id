<h2>🧐 Features</h2>

Here're some of the project's best features:

*   JSON API
*   Website design with EJS
*   light and fast
*   colored console
*   cors allowed
*   logs the requests
*   compatible with discord discriminator removal update

  
  
<h2>💻 Built with</h2>

Technologies used in the project:

*   fs
*   express
*   discord.js@13
*   consoled.js@1.0.4
*   ejs
*   cors
